# Haunted Town - Monsters.txt
# Format (13 columns, comma-separated):
# col  1 - monsterClass     : Java subclass name to instantiate
# col  2 - name             : display name shown to player
# col  3 - level            : integer level (1-4)
# col  4 - maxHp            : level * 25 (pre-calculated)
# col  5 - speed            : 1=slow  2=balanced  3=fast
# col  6 - roomId           : SRS room ID where this monster lives
# col  7 - resistType       : MELEE | RANGED | NONE
# col  8 - resistModifier   : negative double  e.g. -0.25 = takes 25% less damage of that type
# col  9 - weakType         : MELEE | RANGED | NONE
# col 10 - weakModifier     : positive double  e.g. +0.25 = takes 25% more damage of that type
# col 11 - description      : flavour text for journal and size-up display (no commas)
# col 12 - abilities        : pipe-separated list. Each ability: name:damagePercent:hitChance:statusEffect:effectChance
#                             damagePercent is fraction of player maxHp (0.20 = 20%)
#                             statusEffect = STUNNED | DISEASED | TERRORIZED | SHACKLED | NONE
#                             effectChance = probability 0.0-1.0 that status applies on hit (0.0 if NONE)
# col 13 - specialFlags     : semicolon-separated key=value pairs for mechanics that need extra data
#                             NONE if no special flags needed
#
# NOTE - Shotgun ability: damagePercent stores the minimum (0.05). SHOTGUN_MAX in specialFlags stores the ceiling (0.30).
#        Code generates a random value between min and max each time Shotgun fires.
# NOTE - Barricade ability: damagePercent=0.00 hitChance=1.00. Always fires. Ends combat (player wins).
#        Code reads HAS_BARRICADE=true flag to trigger door-blocking logic after combat ends.
# NOTE - Evade ability: damagePercent=0.00. Code reads EVADE_PLAYER_HIT_PENALTY to apply -15% to player next hit.
# NOTE - Fortify ability: damagePercent=0.00. Code reads FORTIFY_MELEE_HALF_CHANCE to apply 40% half-damage chance.
# NOTE - Darken ability: damagePercent=0.00 hitChance=0.40 (activation roll). Code reads DARKEN_MULTIPLIER=1.5.
# NOTE - Flee ability (Freak): hitChance=0.00 at base. Code reads FLEE_HP_THRESHOLD and FLEE_CHANCE_WHEN_LOW.
# NOTE - Shadow NIGHT_DOUBLE_HIT passive: code reads flag and doubles all hitChance values if isNight==true.
# NOTE - "Small Processor-1" in SRS Bell Tower room table is a typo for "Small Possessor-1" --
#        no monster class called Processor exists in the SRS monster table.
#        Two SmallPossessor instances (GY-01 and CH-03) is intentional per SRS room assignments.
#
